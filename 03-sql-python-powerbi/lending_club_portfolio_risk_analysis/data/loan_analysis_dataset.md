# Data Folder Notice

## English Version

The original and processed datasets used in this project are **not included in this repository** due to GitHub file size limitations (maximum 100 MB per file).

The Lending Club dataset used in this analysis exceeds this limit, therefore:

- Raw data files are stored locally
- Processed datasets generated through SQL and Python are also stored locally
- The repository includes all transformation logic, SQL scripts, and documentation required to reproduce the analysis

To replicate this project:

1. Download the Lending Club dataset from Kaggle.
2. Place the dataset inside the `/data/` directory.
3. Execute the Python and SQL scripts in the order described in the project documentation.

This approach follows professional data portfolio standards, where:
- Analytical logic and reproducibility are fully documented
- Business insights and decision-making process remain transparent
